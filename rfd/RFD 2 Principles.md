# Principles

## Mission

Build a framework for robotics that we trust the company's future to — and that others can trust too.

## Values

### Robustness

To operate a large number of machines with high availability, the core system
must be fault tolerant and reliable. Nodal achieves this by being thoroughly
tested and engineered to fail in predictable, controlled ways rather than in
whatever manner is most expedient.

### Simplicity

For systems to be truly comprehensible, the platform it relies on must embody
the same quality. Nodal is simple because it strives to be as small as possible
and avoid the accidental complexity that is common in alternatives.

### Operability

The system must be straightforward to inspect at runtime and not depend on any
large amount of custom tools that introduce unecessary complexity. Nodal is
well-suited for operational environments because many of its features were inspired
by systems thata lready see widespread adoption in production.
